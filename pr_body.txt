💡 What
Added a red asterisk (`*`) to the label of required fields in the "Add Task" and "Create new list" forms.

🎯 Why
This micro-interaction gives users immediate visual feedback on which fields must be filled out before submitting the form, preventing validation errors on submit and reducing frustration.

📸 Before/After
Before: The "Task Name", "List Name", and "Emoji" fields had no visual indication that they were mandatory.
After: These fields now feature a red `*` next to their label. See `add_list_modal.png` and `add_task_form.png` screenshots inside `/home/jules/verification/screenshots`.

♿ Accessibility
The visual indicator (`*`) is wrapped in a `span` with `aria-hidden="true"`. Because the underlying `Input` components already utilize the native HTML5 `required` attribute (which screen readers handle automatically), this approach correctly provides a visual cue for sighted users without duplicating announcements ("required star required") for screen reader users.
